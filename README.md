# Cereal Filament Drybox Bearing Mount

This project contains 3D-printable parts to convert common cereal dispenser containers into single-spool filament dryboxes.

## Bill of Materials (BOM)

The following parts are required (unless denoted as optional) to convert one (1) cereal dispenser into a filament drybox.

| Name | Quantity | Link | Notes |
| - | - | - | - |
| Bearing Mount (STL) | 2 | Printed part | Choose the appropriate bearing mount for your heat set insert size (see below) |
| Bearing Mount Plate (STL) | 2 | Printed part | |
| M3x0.5 Heat Set Inserts | 6 | 5.6mm OD: https://amzn.to/3HfA4ty; 5mm OD: https://www.aliexpress.com/item/4000232858343.html | Use either 5.6mm x 3.8mm OD (commonly available on Amazon US) or 5mm OD x 4mm (Voron style) |
| M3x05 8mm Socket Head Cap Screws | 6 | https://amzn.to/3HbcQos | Alternatively, purchase from Bolt Depot (https://www.boltdepot.com/Product-Details.aspx?product=22469 or https://www.boltdepot.com/Product-Details.aspx?product=23064) |
| 608ZZ Skateboard Bearings | 2 | https://amzn.to/3HgmXII | |
